# NexLoan — Personal Loan Origination System

An AI-first Personal Loan lifecycle prototype built as a single-page application.

## Features
- Inquiry & Lead Generation
- Full Application + KYC Upload
- AI Underwriting Engine (DTI + Credit Score)
- Loan Disbursement Simulation
- EMI Servicing & Schedule
- Pre-closure Calculator
- No-Dues Certificate Generator
- Theoremlabs Credit Score Algorithm
- AI OCR Document Engine
- Identity Fraud Detection
- In-chat Loan Status with OTP Login

## Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Anthropic Claude API (claude-sonnet-4-20250514)

## How to Run
Just open `index.html` in any browser.
